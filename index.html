Card Game System

Card Game System
Card/Deck Creation System
System that uses a graphical interface to allow players to manipulate cards on a virtual board, made using pure Javascript, HTML, and CSS. Card information and positions of cards on the board are stored in a MongoDB database, allowing users to connect with other players to play a game. Additionally, there is a system for users to create their own cards and decks, similarly using a graphical interface.
